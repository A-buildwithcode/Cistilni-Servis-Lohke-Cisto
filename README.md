# Lohke Čisto - Spletna stran

Profesionalna spletna stran za čistilni servis Lohke Čisto.

## Funkcionalnosti

- ✅ Moderna, uglajena spletna stran z animacijami
- ✅ Responsive dizajn za vse naprave
- ✅ Kontaktna forma za povpraševanja
- ✅ Avtomatsko pošiljanje emailov preko EmailJS
- ✅ Profesionalen dizajn z gradienti in animacijami

## Moj Projekt
To je moj prvi spletni projekt, kjer sem ustvarila interaktivno spletno stran z uporabo html, css in JavaScript.
Vključuje obrazec za povpraševanje preko Formspree in je naložena na GitHub pages. Stran sem dodala tudi na Google Search Console.

## Uporabljene tehnologije:
-HTML
-CSS
-JavaScript
-Formspree
-GitHub Pages
-Google Search Console
